# calendar-privacity

Esta aplicación respeta la privacidad de sus usuarios.
No recopilamos información personal identificable de forma directa.

Podemos utilizar servicios de terceros para mejorar la experiencia del usuario, como herramientas de análisis o publicidad, que pueden recopilar información anónima.

No compartimos información personal con terceros.
